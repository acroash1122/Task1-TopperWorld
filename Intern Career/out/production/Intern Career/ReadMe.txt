# Hangman Game

This is a simple Hangman game implemented in Java with a graphical user interface (GUI) using Swing.
The game randomly selects a word from a predefined list, and the player has to guess the word by entering one letter at a time.

## Features
- Randomly selects a word from a predefined list of words.
- Graphical user interface using Swing.
- Tracks the number of incorrect guesses (misses).
- Allows the player to start a new game.

## Requirements

- Java Development Kit (JDK) 8 or higher
- IntelliJ IDEA (or any other Java IDE)

## How to Run

1. :Clone the repository:
    ```ssh
    git clone https://github.com/acroash1122/hangman-game.git
    ```

2. **Open the project in IntelliJ IDEA**:
    - Open IntelliJ IDEA.
    - Click on `File` > `Open...`.
    - Navigate to the directory where you cloned the repository and select it.

3. **Run the application**:
    - Locate the `HangmanGUI.java` file in the `src` directory.
    - Right-click on the file and select `Run 'HangmanGUI.main()'`.

## Usage

- Enter a single letter in the text field and click the "Guess" button to make a guess.
- The game will display the current state of the word with guessed letters revealed.
- Incorrect guesses will be counted as misses.
- Click the "New Game" button to start a new game.
